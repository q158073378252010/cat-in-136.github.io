見よ！これが M+ font と CSS font-weight の実力だ！

Text by: cat_in_136
Date: 2012-04-07T12:30:00+09:00 (Modified: 2014-11-10T00:02:35+09:00)
Tags: Font Css

太字と標準しかない他のゴシック体のフリーフォントとは気合が違うんだよ！

あなたのブラウザでの表示（予め mplus-1p-*.ttf をすべてインストールしておく必要があります。）

font-weight: 200; 新しい朝が来た希望の朝。
font-weight: 300; 新しい朝が来た希望の朝。
font-weight: 400; 新しい朝が来た希望の朝。

font-weight: 800; 新しい朝が来た希望の朝。
font-weight: 900; 新しい朝が来た希望の朝。
