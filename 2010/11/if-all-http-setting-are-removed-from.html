If all HTTP setting are removed from Configuration Mania...

Text by: cat_in_136
Date: 2010-11-09T01:14:00+09:00 (Modified: 2014-11-10T00:02:32+09:00)
Tags: Configuration mania Security Mozilla

I tried removing all HTTP setting from Configuration Mania (ref Mozilla Add-on validation system says HTTP setting is generally not allowed.). I remove "HTTP Network" setting panel and Security->Referrer.

I commit new branch "http_setting_deleted". You can download the experimental XPI file configuration_mania_-TEST-http_setting_deleted-1.12.2010100811-ab2047b19771.xpi from the download page. If you wanna configure unsafe HTTP settings, you SHOULD NOT install this XPI.
