Configuration Mania 1.11.2009021401

2010-02-14T13:47:00+09:00

Configuration Mania 1.11.2009021401 has been submitted to AMO. It's in the sandbox now (i.e. waiting for review).

  1. add pl and zh-CN locales
  2. add "Browser"->"TabBrowsing"->"Enable thumbnail preview (Firefox3.6+)"->"on Taskbar (Windows7)" (browser.taskbar.previews.enable)
  3. fix "Popup Options Resetting" bug

On Babelzilla sites, fr and de-DE locales are also available, but their progress levels are still low. So these locales aren't included.
