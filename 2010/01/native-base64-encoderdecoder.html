Native base64 encoder/decoder

Text by: cat_in_136
Date: 2010-01-31T23:50:00+09:00 (Modified: 2010-02-20T11:41:27+09:00)
Tags: Javascript

window.atob() decodes a base64-encoded string, and window.btoa() creates a base64-encoded string from binary data. window.btoa() is useful for creating data: URI.

atob() and btoa() work on Firefox 2+, safari 3.1+, Opera and Google chrome.
